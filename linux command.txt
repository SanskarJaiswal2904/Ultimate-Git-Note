ls - The most frequently used command in Linux to list directories
pwd - Print working directory command in Linux
cd - Linux command to navigate through directories
mkdir - Command used to create directories in Linux
cd -   - To move out of a folder one time
cd --   - To move out of folder two times
touch <file name> -> To create a file

clear - Clear the terminal display
mv - Move or rename files in Linux 		"mv<file><name of new file>"
cp - create a seperate copy of file in linux 	"cp<file><name of copy file>"
rm <file name>- The rm command is used to delete files and folders
rm -rf temp.txt  - To remove a folder name temp.txt

zip - Zip files in Linux
unzip - Unzip files in Linux

ifconfig - Display network interfaces and IP addresses
